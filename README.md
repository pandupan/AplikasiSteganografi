# 🔐 Aplikasi Steganografi

Aplikasi **steganografi** untuk menyembunyikan dan mengekstrak pesan rahasia di dalam media gambar. Menggabungkan konsep kriptografi dan steganografi untuk keamanan data ganda.

## ✨ Fitur

- **Penyisipan Pesan** — Sembunyikan teks rahasia ke dalam gambar
- **Ekstraksi Pesan** — Baca pesan tersembunyi dari gambar
- **Keamanan Ganda** — Kombinasi steganografi dengan enkripsi

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- Library pemrosesan gambar (Canvas API)

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
